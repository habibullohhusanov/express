# Express template

## Use
**Download the template**
### Enter the terminal
```
npm install
```
**Rename the .env.example file to .env .**
<br/>
**Enter a base link .**

### run developer mode
```
npm run dev
```
### run template
```
npm start
```

**The template runs on port 5000 at http://localhost .  http://localhost:5000**
